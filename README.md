 # 🚀 Development of a Spam Email Detection and Classification System Using Supervised Learning Models

An Artificial Intelligence-based Spam Email Detection and Classification System developed using **Supervised Machine Learning Models** to automatically classify emails as **Spam** or **Ham (Legitimate)**. The system helps users identify unwanted emails, improve email security, and reduce phishing risks through intelligent text classification.

---

# 📌 Project Overview

This project uses a **Multinomial Naive Bayes Classifier** with **TF-IDF Vectorization** to analyze email text and classify messages as **Spam** or **Ham**. The Machine Learning model is developed using **Google Colab**, while the frontend dashboard is built using **HTML5, CSS3, and JavaScript**. The complete project is managed using **GitHub** and deployed through **Vercel**.
